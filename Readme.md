# 프로젝트 이름

## 설명
소스 폴더와 선택적으로 로그 폴더를 인자로 받아 처리하는 프로그램입니다. 로그 폴더는 제공하지 않으면 현재 디렉토리가 기본값입니다.

## 설치

### 1. 가상 환경 만들기
가상 환경을 만듭니다:

```bash
python -m venv venv
```

### 2. 가상 환경 활성화
- **Windows**:
  ```bash
  venv\Scripts\activate
  ```
- **macOS/Linux**:
  ```bash
  source venv/bin/activate
  ```

### 3. 의존성 설치
다음 명령어로 필요한 라이브러리를 설치합니다:

```bash
pip install -r requirements.txt
```
